- Install npm, sudo apt-get install npm
- sudo apt-get install mongodb
- start mongo sudo /etc/init.d/mongodb start or sudo service mongodb start
- install mongoose, npm install mongoose
- connect to mongo db server, "mongo"
- create database, "use pingpong"
- create api file structure src/controllers, src/routes
- install express, npm install express
- create src/config.js
- npm install jwt-express to handle jwt tokens
- npm install mongoose-bcrypt, does password hashing and comparing
